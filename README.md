# Compiler Unreachable Code Detector

## 📌 Overview
This project is a web-based compiler visualizer that detects unreachable code and demonstrates the working of different compiler phases.

## 🚀 Features
- Detects unreachable code (dead code)
- Supports multiple programming languages
- Visualizes:
  - Lexical Analysis
  - Syntax Analysis (Parse Tree)
  - Semantic Analysis
  - Intermediate Code Generation
  - Code Optimization
  - Final Output

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download the project
2. Open index.html in browser

## 🎯 Example Cases
- Code after return
- Infinite loops (while true)
- If false conditions
- Break/continue unreachable code
